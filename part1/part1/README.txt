To run Webots:
> ros2 launch part1 part_1_launch.py

To run hello_world on the robot:
> ros2 launch part0 hello_world_launch.py

To run obstacle avoidance:
> ros2 launch part1 obstacle_avoidance_launch.py mode:=braitenberg
> ros2 launch part1 obstacle_avoidance_launch.py mode:=rule_based

To run localization:
> ros2 launch part1 localization_launch.py

To run RViz:
> ros2 launch part1 rviz_launch.py
